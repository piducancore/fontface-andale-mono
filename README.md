## Install

```bash
yarn add @piducancore/fonts-andale-mono
```

## Usage

Import the font somewhere in your project like this:

```js
import "@piducancore/fonts-andale-mono"
```

Then, use the font in your styles like this:

```css
font-family: "Andale Mono";
```

That's it.
